# BudgetTripPlanner API

This is the backend for the BudgetTripPlanner application, built with FastAPI.

## Local Setup

1. Clone the repository.
2. Create and activate a virtual environment: `python -m venv venv`
3. Install dependencies: `pip install -r requirements.txt` (We will create this file later)
4. Run the development server: `uvicorn app.main:app --reload`
